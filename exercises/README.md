# Exercises

This folder is where you put your work.

## Your Task

Create a file in this folder called **`about-me.md`**.

Use the template in `about-me-template.md` as a starting point — copy its contents into your new file and fill it in with your own information.

### Requirements

Your `about-me.md` file must:

1. Be named exactly `about-me.md` (lowercase, with a hyphen)
2. Be saved inside this `exercises/` folder
3. Contain a `## About Me` heading
4. Have at least a few lines of content about yourself

### After Creating the File

Follow the instructions in [Step 5](../steps/05-create-file.md) to stage and then commit the file.

---

> **File path reminder:** The automated check looks for the file at exactly `exercises/about-me.md`. Make sure you do not accidentally save it to the root of the repo or spell the name differently.
